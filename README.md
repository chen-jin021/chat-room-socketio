# Chat Room Application with SocketIO

This is a chat room application built with SocketIO, Javascript and HTML. Users can join the chat room and send messages to each other in real time.

## Getting Started

To run this application on your local machine, follow these steps:

1. Clone this repository to your local machine.
2. Make sure you have Node.js and npm installed on your machine.
3. Install the required dependencies by running `npm install` in your terminal.
4. Start the server by running `node server.js`.
5. Open a browser and navigate to `http://localhost:3000`.

## Usage

To use this application, simply enter a username and join the chat room. Once you are in the chat room, you can send messages to other users by typing in the message input box and pressing the `Send` button or hitting `Enter`.

## Dependencies

This application depends on the following packages:

- [SocketIO](https://socket.io/)
- [Express](https://expressjs.com/)

These dependencies are installed automatically when you run `npm install`.
